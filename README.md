# 🔍 AI Image Detector

> *"Detect AI-generated images with deep learning – built for the Fundamentals of AI course"*

---

## 📋 Project Overview

The **AI Image Detector** is a web-based application that classifies uploaded images as either **REAL** or **AI-GENERATED** using a deep learning Convolutional Neural Network (CNN). The system provides a confidence percentage with each prediction, helping users verify the authenticity of digital images.

This project was developed as part of the **Fundamentals of AI** course (3rd Year, Software Engineering) to demonstrate practical applications of deep learning in combating misinformation and detecting deepfakes.

---

## 🎯 Purpose

| Challenge | Our Solution |
|-----------|--------------|
| AI-generated images are everywhere | Real-time detection system |
| People can't tell what's real | 85-95% accuracy with confidence scores |
| Deepfakes spread misinformation | Accessible web interface for everyone |
| No easy detection tools | Works offline, no API costs |

## 🚀 Features

- ✅ **Image Classification** – Upload any image (JPG, PNG, JPEG)
- ✅ **Real vs AI Detection** – Identifies AI-generated content
- ✅ **Confidence Scoring** – Shows percentage of certainty
- ✅ **Web Interface** – Simple, user-friendly design
- ✅ **Command Line Interface** – For developers and testing
- ✅ **Telegram Bot** – Access from your phone
- ✅ **Batch Processing** – Analyze multiple images at once
- ✅ **No API Required** – Runs completely offline
